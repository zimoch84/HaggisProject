# Wymagania srodowiska

## Narzedzia

- .NET SDK 8.0+
- Docker Desktop (z Compose)
- PowerShell 7+ lub Windows PowerShell
- Git

## Opcjonalnie (UI Flutter)

- Flutter SDK
- Android Studio / Xcode / Chrome (w zaleznosci od targetu)

## Porty domyslne

- API: `8080`
- SQL Server: `1433`

## Uwagi

- Upewnij sie, ze porty `8080` i `1433` nie sa zajete.
- Dane logowania i connection stringi trzymaj finalnie w zmiennych srodowiskowych lub secret store.
